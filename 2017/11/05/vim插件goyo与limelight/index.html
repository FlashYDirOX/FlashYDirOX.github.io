vim插件goyo与limelight

  • goyo和limelight两款vim插件使得vim变得更加适合阅读与写作,同时也可以美化vim的界面

  • 这是来自官网的截图

安装

  • 我使用的是vim-plug插件，于是只需要将这两条插件管理语句添加到vimrc中，vundle的使用也类似

配置与使用

  • 输入Goyo打开Goyo，输入Limelight打开limelight，输入Goyo!则关闭,Limelight!同理是关闭

  • 这样输入两次是不是很烦？那么可以试着在打开Goyo的时候自动打开Limelight,同时将打开Goyo映射为一个快捷键，下面给出我的配置

注意

  • 如果你正在使用的是limelight不支持的配色，那么在打开limelight的时候就会报错，这时候就需要根据你的配色是ANSI code还是RGB color来调整
